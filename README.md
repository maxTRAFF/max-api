## О приложении

MaxTraff объединяет аккаунты, прокси, инструменты, процессы, статистику и результаты в одном рабочем интерфейсе.

Это не бот, не веб-панель и не набор скриптов, а самостоятельное приложение для ежедневной работы оператора.

## Возможности

| Раздел | Что внутри |
| --- | --- |
| Аудитория | Инвайт в чаты и каналы, работа по ID, накопление базы |
| Сообщения | Массовая отправка, вложения, задержки, статусы выполнения |
| Парсинг | Сбор данных из чатов, импорт списков, сохранение результатов |
| Проверки | Проверка номеров и валидация данных перед запуском |
| Аккаунты | Добавление, авторизация, импорт и экспорт сессий |
| Прокси | Импорт, привязка к аккаунтам и проверка доступности |
| Процессы | Логи, статусы, история запусков и анализ ошибок |
| Статистика | Дашборд с успешностью, ошибками и объемом действий |

## Для кого

MaxTraff подойдет тем, кто хочет:

- масштабировать работу с аккаунтами;
- привлекать аудиторию в чаты и каналы;
- запускать повторяющиеся действия быстрее;
- работать с большими списками пользователей;
- контролировать прокси, процессы и результаты;
- использовать один инструмент вместо десятков файлов и скриптов.

## Платформы

Приложение разрабатывается как полноценный desktop-инструмент и, в зависимости от сборки, может использоваться на:

- Windows;
- macOS;
- Linux.

## Доступ

MaxTraff является закрытым приложением с доступом по подписке.

Для работы требуется ключ активации. После активации открываются основные разделы приложения.

## Первый запуск

При первом запуске Windows или macOS может показать предупреждение о неизвестном разработчике. Это нормально для приватных приложений, которые распространяются напрямую, а не через публичные магазины.

**Windows:** нажмите `Подробнее`, затем `Выполнить в любом случае`. Обычно с Windows нет проблем и кроме уведолмения о том, что приложение загруженно от неизвестных разрабочтиков других неудобств нет!

## Локальные данные

MaxTraff хранит рабочие данные на компьютере пользователя: списки, результаты парсинга и проверок, историю процессов, экспортированные файлы и локальные данные приложения.

## Итог

MaxTraff помогает держать под контролем аккаунты, прокси, инвайты, сообщения, парсинг, проверки, процессы и статистику в одном приложении.

Главный фокус: скорость, порядок и понятный результат.
